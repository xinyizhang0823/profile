<section id="overview">
  <div class="section-label"><span>01 &nbsp;问题定义</span></div>
  <h2>我们在解决什么问题？</h2>
  <p class="section-desc">情绪类可穿戴产品流失率远高于预期——用户并非不想记录情绪，而是现有产品无法准确"接住"他们想表达的感受。</p>
  <div class="problem-grid">
    <div class="problem-card p1">
      <div class="problem-num">痛点 01 · 系统侧</div>
      <h3>自动识别不准，<br>破坏用户信任</h3>
      <p>系统推断的情绪与真实感受频繁不符，引发认知失调，用户停止主动校正数据。</p>
      <div class="quote-block">"它说我有压力，但我当时只是走路快了点。"——访谈 P3</div>
    </div>
    <div class="problem-card p2">
      <div class="problem-num">痛点 02 · 语义侧</div>
      <h3>标签颗粒度不足，<br>表达精度受限</h3>
      <p>语义重叠（"焦虑"vs"不安"）+ 颗粒度粗糙，导致用户在相似词间反复犹豫，记录意愿下降。</p>
      <div class="quote-block">"想记那种平静但有点空的感觉，但选项里没有。"——访谈 P1</div>
    </div>
  </div>
</section>

// 滚动动画：使用 Intersection Observer 实现进入视口时淡入上浮
document.addEventListener('DOMContentLoaded', function() {
  const observerOptions = {
    root: null, // 视口
    rootMargin: '0px',
    threshold: 0.1 // 元素进入视口10%时触发
  };

  const observer = new IntersectionObserver(function(entries, observer) {
    entries.forEach(function(entry) {
      if (entry.isIntersecting) {
        entry.target.classList.add('visible');
        observer.unobserve(entry.target); // 只触发一次
      }
    });
  }, observerOptions);

  // 观察所有 section 元素
  const sections = document.querySelectorAll('section');
  sections.forEach(function(section) {
    observer.observe(section);
  });
});
